# Prompt variants

This folder contains the three prompt variants used in the grid experiments reported in the paper.

- [p01.txt](p01.txt): shorter accessibility-oriented prompt
- [p02.txt](p02.txt): accessibility-oriented prompt focused on scene purpose, objects, actions, spatial relations, and visible text
- [p03.txt](p03.txt): more constrained professional audiodescription prompt with explicit stylistic and structural guidance

Mapping used in the paper:
- `p01` -> [p01.txt](p01.txt)
- `p02` -> [p02.txt](p02.txt)
- `p03` -> [p03.txt](p03.txt)
