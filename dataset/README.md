# Dataset

The repository includes the dataset used in the paper under [UFPE-Scenes](UFPE-Scenes).

Contents:
- `UFPE-Scenes/images`: input images
- `UFPE-Scenes/annotations`: Brazilian Portuguese reference descriptions
- `UFPE-Scenes/annotations_en`: English translations used for some evaluation metrics

The dataset is used as a controlled benchmark for assistive image description in Brazilian Portuguese.
